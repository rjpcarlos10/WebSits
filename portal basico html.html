        <h3>Sintaxis:</h3>

        <p>Los enlaces en HTML permiten navegar entre páginas.</p>
        <h3>Sintaxis:</h3>
        <pre><code><a href="URL">Texto del enlace</a></code></pre>
        <h3>Ejemplo:</h3>
        <pre><code><a href="https://www.google.com">Ir a Google</a></code></pre>
    </section>

    <section>
        <h2>4. Hyperlinks</h2>
        <p>Son una de las funcionalidades principales de HTML. Permiten vincular recursos externos o internos.</p>
        <h3>Uso:</h3>
        <p>Se utilizan para navegación entre páginas o archivos.</p>

    <section>
        <h2>5. CSS Text</h2>
        <p>CSS permite personalizar el texto con propiedades como alineación, transformación, espacio y más.</p>
        <h3>Ejemplo:</h3>
        <pre><code>p {
    text-align: center;
    text-transform: uppercase;
    letter-spacing: 2px;
}</code></pre>
    </section>

    <section>
        <h2>6. CSS Font</h2>
        <p>Controla el tamaño, estilo y tipo de fuente en una página web.</p>
        <h3>Ejemplo:</h3>
        <pre><code>h1 {
    font-family: 'Arial', sans-serif;
    font-size: 24px;
    font-weight: bold;
}</code></pre>
    </section>

    <section>
        <h2>7. HTML Styles - CSS</h2>
        <p>Los estilos pueden aplicarse directamente en el HTML mediante el atributo <code>style</code>.</p>
        <h3>Ejemplo:</h3>
        <pre><code><p style="color: blue; font-size: 16px;">Texto estilizado</p></code></pre>
    </section>

    <section>
        <h2>8. CSS Syntax - CSS</h2>
        <p>La sintaxis CSS utiliza selectores para apuntar elementos HTML y propiedades para definir estilos.</p>

        <h2>9. Selectors</h2>
        <p>Los selectores en CSS identifican elementos HTML para aplicarles estilos específicos.</p>
        <h3>Ejemplo:</h3>
        <pre><code>/* Selector de clase */
.button {
    background-color: blue;
    color: white;
}

/* Selector de ID */
#header {
    background-color: green;
}</code></pre>
